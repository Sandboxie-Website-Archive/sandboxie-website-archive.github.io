Post by OldGrantonian » Sat Feb 24, 2018 6:30 am

If I want to run a program within SB, and then change my mind, I don't think there's any elegant way to back out. AFAIK, I must continue to launch the program and then close the program.

Here's an example using Chrome.

- Start Chrome as follows: R-click on desktop icon > Run Sandboxed
- The small "Run Sandboxed" dialog appears. Click OK.
- The bigger "Sandboxie" dialog appears.

And now, there's no way to back out. The "x" at top right does not respond.
AFAIK, the only option now is to click "Click to close this window and continue"

Does anyone know how to back out of this process?
