Re: Slow loading of the starting page with Firefox 58.0.1 [SOLVED]

Posted: Tue Jan 30, 2018 5:14 pm
by Peter 123
Hello Barb and thanks for all your support.

The advice from the first linked thread did not help. (Reducing the value for "security.sandbox.content.level" brought back the sound but did not fix the issue with the delay in loading the starting page.)

Nevertheless I found (indirectly) the solution in another thread linked by APMichael.
It's this thread:
viewtopic.php?f=11&t=25364

I followed the following advice given there by bo.elam and by Geordy (originally also for the sound problem):

Again in about:config of Firefox (opening it outside the sandbox) I changed the value for browser.tabs.remote.autostart from "true" to "false". (The value for "browser.tabs.remote.autostart.2" should already be on "false" by default.)

After this change Firefox starts again within the Sandbox as it did before the update to version 58.0.1. (That means: with essentially no delay.)

Many thanks to APMichael (for the link) and especially to bo.elam and Geordy (for the solution).

PS:
@ Barb:
Yes, I have nvidia drivers installed. I (hope I) will keep in mind the other thread you mentioned, in case that there is a n new problem with a delay.

According with the above described issues I tested with 2 scenarios concerning the modifications in about:config

Scenario A:
- security.sandbox.content.level: value reduced from 4 (= default) to 2

Here a specific situation with which I tested Scenario B when activating the Resource Access Monitor:
- Starting the browser (within Sandboxie) --> the starting page (google.at) opens without delay ---> from my bookmarks I choose this forum (forums.sandboxie.com/phpBB3) ---> the forum page opens also without delay ---> I click on one of the links in the list "LAST POST" ---> after clicking the browser hangs/freezes and only after some seconds the post opens. - This seems to happen only one time. Afterwards using the browser (staying either on the same page or going to any other one) seems to work smoothly, without any further delay.
